# Support

Thanks for trying Resume Studio.

## Bugs

Use GitHub Issues for non-sensitive bug reports.

## Feature Requests

Use GitHub Issues for feature requests and product ideas.

## Import or Export Problems

Use GitHub Issues for import/export problems, including DOCX, PDF import, Markdown import, backup restore, DOCX export, Markdown export, and Print / Save as PDF issues.

## Private Feedback

Email labs@bractos.com if your feedback includes private context or sensitive details.

## Security Vulnerabilities

Email labs@bractos.com for security vulnerabilities.

Do not file a public GitHub issue for security vulnerabilities.

## Privacy Reminder

Do not post private resume content, personal contact information, job application details, or sensitive data in public issues, pull requests, screenshots, fixtures, or examples.
